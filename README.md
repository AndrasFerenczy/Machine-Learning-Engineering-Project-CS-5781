# Minitorch

Implementation of the [Minitorch](https://minitorch.github.io) library for the Machine Learning Engineering course at Cornell University.

Minitorch is a diy 
teaching library for machine learning engineers who wish to 
learn about the internal concepts underlying deep learning 
systems. It is a pure **Python re-implementation of the Torch 
API** designed to be simple, easy-to-read, tested, and incremental. 
The final library can run Torch code.

1. module: [ML Programming Foundations](https://minitorch.github.io/module0/module0/)
2. module: [Autodifferentiation](https://minitorch.github.io/module1/module1/)
3. module: [Tensors](https://minitorch.github.io/module2/module2/)
4. module: [GPUs and Parallel Programming](https://minitorch.github.io/module3/module3/)
5. module: [Foundational Deep Learning](https://minitorch.github.io/module4/module4/)
